# GNSS 电离层 nc 数据绘图程序需求文档

## 1. 项目目标
开发一个 Python 程序，用于读取已下载的 VTEC、dTEC、ROTI 的 `.nc` 文件，自动生成适用于毕业论文的电离层图像，并支持单时刻绘图、连续时序绘图、批量导出和事件对比分析。

本程序重点服务于地磁暴期间赤道等离子体泡（EPB）及相关电离层扰动结构的可视化分析。

---

## 2. 输入数据

### 2.1 数据类型
输入为 3 类 netCDF 文件：
- VTEC
- dTEC
- ROTI

### 2.2 文件要求
- 文件格式：`.nc`
- 文件命名：保持原始文件名，不强制改名
- 支持单文件读取和文件夹批量读取
- 默认每个文件包含：
  - 时间维
  - 纬度维
  - 经度维
  - 对应变量值

### 2.3 数据来源
用户已提前下载好 nc 文件，程序只负责读取、处理和绘图，不负责下载。

---

## 3. 功能需求

## 3.1 数据读取
程序应能：
1. 自动识别 nc 文件中的时间、纬度、经度、变量名
2. 若变量名不统一，允许设置候选变量名列表自动匹配
3. 读取单时刻二维网格数据
4. 支持遍历一个文件内所有时刻
5. 支持遍历多个文件并按时间排序

---

## 3.2 数据预处理
程序应具备以下基础处理能力：
1. 识别缺测值（如 NaN、FillValue）
2. 将缺测值屏蔽，不参与绘图
3. 支持经度范围转换：
   - 0~360
   - -180~180
4. 支持按区域裁剪：
   - 全球
   - 亚洲-太平洋区
   - 南美区
   - 自定义经纬度范围
5. 支持统一颜色范围，便于不同时间对比
6. 支持时间格式转换为 UTC 字符串用于标题和文件名

---

## 3.3 绘图功能

### 3.3.1 单张地图绘图
可绘制某一时刻的：
- VTEC 分布图
- dTEC 分布图
- ROTI 分布图

要求：
- 使用经纬度投影地图
- 带海岸线、国界、经纬网
- 带颜色条
- 标题包含：
  - 数据类型
  - 日期
  - UTC 时间
- 图片分辨率适合论文插图

### 3.3.2 批量时序绘图
对某个文件内所有时刻自动输出图片序列：
- 每个时刻一张图
- 文件名按时间顺序命名
- 输出到指定文件夹

### 3.3.3 对比图绘制
支持以下对比方式：
1. 同一时刻 VTEC / dTEC / ROTI 分开绘制
2. 同一时刻 3 幅拼图
3. 同一天不同时刻拼图
4. 不同事件日同一时刻对比图

### 3.3.4 动图生成（可选）
支持将批量输出图片合成为：
- GIF
- MP4

---

## 3.4 输出结果
程序输出内容包括：

### 3.4.1 静态图片
格式：
- PNG（默认）
- JPG（可选）

### 3.4.2 动图
格式：
- GIF
- MP4

### 3.4.3 日志信息
终端输出：
- 当前读取文件
- 当前处理时刻
- 输出路径
- 异常警告信息

---

## 4. 图像规范

## 4.1 通用规范
- 图片尺寸统一
- 字体清晰
- 颜色条范围固定或可配置
- 标题规范统一
- 坐标轴显示经纬度
- 海岸线清晰

## 4.2 建议配色
- VTEC：连续色带，突出高低值分布
- dTEC：发散色带，突出正负异常
- ROTI：高值突出色带，强调不规则体活动区

## 4.3 图题格式
示例：
- `VTEC Map 2024-05-12 10:00 UTC`
- `dTEC Map 2024-05-12 10:00 UTC`
- `ROTI Map 2024-05-12 10:00 UTC`

---

## 5. 程序结构要求

建议拆分为以下模块：

### 5.1 `reader.py`
负责：
- 读取 nc 文件
- 识别变量
- 返回时间、纬度、经度、数据数组

### 5.2 `preprocess.py`
负责：
- 缺测值处理
- 经纬度转换
- 区域裁剪
- 时间格式化

### 5.3 `plotter.py`
负责：
- 单图绘制
- 拼图绘制
- 配色和地图元素设置

### 5.4 `batch_export.py`
负责：
- 批量遍历时刻
- 批量输出图片
- 合成 GIF/MP4

### 5.5 `main.py`
负责：
- 参数配置
- 调用各模块执行任务

---

## 6. 配置需求
程序应尽量将可修改参数集中到配置区，包括：
- 输入文件夹路径
- 输出文件夹路径
- 数据类型（VTEC/dTEC/ROTI）
- 绘图区域
- 时间索引
- 色标范围
- 是否生成动图
- 输出 dpi
- 图片格式

1. 文件不存在
2. 文件夹为空
3. nc 文件无法读取
4. 未识别到时间/纬度/经度/变量
5. 某时刻数据为空
6. 输出目录无权限创建
